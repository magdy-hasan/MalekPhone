###Featurs testing video [here](https://youtu.be/lDyP7cEwBQg).

Featurs:

    -View all products
    
    -Add new category
    
    -View specific category
    
    -Add new product
    
    -Edit category/product information
    
    -Add/Edit image for products
  
  Other features to be implemented
